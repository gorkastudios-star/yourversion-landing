# YourVersion Landing Page

Premium one-page landing website for "YourVersion", a premium cover band in Bilbao.

## How to Run Locally

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.

## Deployment on GitHub Pages

1. Push this folder to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select the branch (e.g., `main`) and folder `/ (root)`.
4. Save and your site will be live at `https://<username>.github.io/<repo-name>/`.

## Customization Guide

- **Quote Form**: The form generates a summary and uses a `mailto:` fallback. Edit the email address in `app.js`.
- **Video**: Replace the placeholder in `index.html` with your actual YouTube embed link.
- **Reviews**: Modify the testimonial cards in `index.html`.
- **Repertoire**: Update the song list in the "Repertorio" section of `index.html`.
- **Social Links**: Update Instagram and WhatsApp links in the footer of `index.html`.
